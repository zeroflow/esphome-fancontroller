Detailed technical documentation for hardware revisions, firmware modules, and configuration options.

## Sections

### [Hardware Revisions](/reference/hardware/)

Pin assignments, board specifications, and revision comparison for every WiFi Fan Controller version from Rev 1.0 through Rev 3.3. Includes firmware installation instructions per board revision.

### [Modules](/reference/modules/)

Optional add-on packages for temperature control (PID, linear, curve), closed-loop RPM regulation, and RGB LED status indicators. Each module page documents configuration variables, Home Assistant entities, and usage examples.
